#define TABLE_MAX_LOAD   0.75

The max load factor of the hashtable.

If the max load factor multiplied with the capacity is reached we grow the hashtable

Adjusts the capicity of a hashtable.

Parameters
tableThe hashtable where the capacity is changed
capacityThe new capacity of the hashtable

We grow the hashtable when it becomes 75% is filled, so we can wrap around the entries when we look for a key, without risking an infinite loop when the hashtable is full.
