◆ object_hash_string()

static uint32_t object_hash_string ( char const *  key,
uint32_t  length 
)
static

FNV-1a hash function.

Fownler-Noll-Vo is a non-cryptographic hash function, that comes in three different version FNV-0, FNV-1 and FNV-1a. There are 32-, 64-, 128-, 256-, 512-, and 1024-bit variants of the function. We use the 32-bit variant to hash all the strings in cellox. The current implementation is called fowler-noll-vo pippip yurii − https://www.codeproject.com/articles/716530/fastest-hash-function-for-table-lookups-in-c

Parameters
keyThe key that is hashed
lengthThe length of the key
Returns
The hashvalue of the key
