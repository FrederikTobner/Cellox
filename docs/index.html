Cellox

Introduction

Welcome to the technical documentation of the cellox interpreter. Cellox is a dynamically typed, object oriented, general purpose, high-level scripting language that offers automatic memory management to the programmer.
It is based on the book crafting iterpreters by Robert Nystrom.
