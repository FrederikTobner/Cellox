FNV-1a hash function

Fownler-Noll-Vo is a non-cryptographic hash function, that comes in three different version FNV-0, FNV-1 and FNV-1a. There are 32-, 64-, 128-, 256-, 512-, and 1024-bit variants of the function. We use the 32-bit variant to hash all the strings in cellox.

Parameters
keyThe key that is hashed
lengthThe length of the key
Returns
The hashvalue of the key
